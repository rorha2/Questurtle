# 변수 이름

lastAttendance

questList

pointNumbers

---

# 함수 이름

동사부터 시작

예시

updateQuest()

givePoint()

usePoint()

loadQuests()

saveQuests()

hideAll()

# Boolean 변수

true / false 의미가 바로 이해되도록 작성

예시

completed

isParent (예정)
